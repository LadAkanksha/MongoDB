----------	Update and Replace Command	----------

 - Modifies an existing document or documents in a collection.

ðŸ’¡ To Update One Row at a Time : 

	UpdateOne() -  Updates a single document within the collection based on the filter.

	db.<collection_nama>.updateOne(
		{field_name : "value"},
		{$set:{updated_field_name1 : "new_value1" , updated_field_name2 : "new_value2"}}
	);

-------------------------------------------------------------------------------------------------------

ðŸ’¡ To Update Multiple Row at a Time :

	updateMany() - Updates all documents that match the specified filter for a collection.

	db.<collection_nama>.updateMany(
		{field_name : "value"},
		{$set:{updated_field_name1 : "new_value1" , updated_field_name2 : "new_value2"}}
	);

-------------------------------------------------------------------------------------------------------

 
ðŸ”§ Field Update Operators

	Operator			Description
    --------------------------------------------------------------------------------------------
	$set			Sets the value of a field. Creates the field if it doesn't exist.
	$unset			Removes the specified field from a document.
	$inc			Increments a field by a specified value.
	$mul			Multiplies a field by a specified value.
	$rename			Renames a field.
	$min			Updates the field only if the specified value is less than the current value.
	$max			Updates the field only if the specified value is greater than the current value.
	$currentDate		Sets the value of a field to the current date.



ðŸ§® Array Update Operators

	Operator			Description
   ----------------------------------------------------------------------------------------------
	$push			Adds an item to an array.
	$pop			Removes the first or last item of an array. (-1 for first, 1 for last)
	$pull			Removes all matching values from an array.
	$addToSet		Adds a value to an array only if it doesnâ€™t already exist.
	$pullAll		Removes all instances of the specified values from an array.
