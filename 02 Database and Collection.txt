----------	Database and Collection		----------

ðŸ”¹ Database
	A database is a container for collections. 
	Each database is stored separately and can contain its own users, permissions, and collections.


ðŸ’¡To Create Database / Use Existing Database : 

	use <database_name>
 
ðŸ’¡To Show Databases :
	
	show dbs

ðŸ’¡To Delete Database :

	db.dropDatabase()

----------------------------------------------------------------------------

ðŸ”¹ Collection
	A collection is a group of MongoDB documents. 
	It is the equivalent of a table in relational databases.

ðŸ’¡To Create Collection : 
	
	db.createCollection("<collection_name>")

ðŸ’¡To Show Collections :
	
	show collections

ðŸ’¡To Rename The Collection :

	db.<collection_name>.renameCollection("<new_collection_name>")

ðŸ’¡To Delete The Collection :

	db.<collection_name>.drop()

-----------------------------------------------------------------------------

ðŸ’¡To List Methods for Database :
	db.help()

ðŸ’¡To List Methods for Collection :
	db.<collection_name>.help()
