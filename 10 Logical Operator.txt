----------		Logical Operator          ----------

	-  In MongoDB, Logical operators to combine or invert query conditions. These are crucial when you need to filter documents based on multiple criteria.

ðŸ§® MongoDB Logical Operators : 

----------------------------------------------------------------------------------------------
|   Operator	                          Description					     |
----------------------------------------------------------------------------------------------	
|    $and		Matches documents that satisfy all the given expressions	     |
|    $or			Matches documents that satisfy at least one of the expressions	     |
|    $nor		Matches documents that fail all the given expressions    	     |
|    $not		Inverts the effect of a query expression		     	     |	
----------------------------------------------------------------------------------------------

School> db.Students.find({$and:[{course:{$in:["MCA","MCS"]}},{age:{$gt:20}}]});
Output =>[
	  {
		_id: ObjectId('68669d375bae57b6af748a62'),
		name: 'Shashikant Yadav',
		age: 22,
		course: 'MCA'
	  },
	  {
		_id: ObjectId('68669d375bae57b6af748a63'),
		name: 'Sushma Gurav',
		age: 21,
		course: 'MCA'
	  }
	]
